Simulation 1

    This application manages the human resources of an intitution.

  1. Firstly, the application allows you to enter basic data into the application in the parameter section which will be used when registering employees.
  2. Secondly to register employees to form a CV for each employer from the system.
  3. Assignment of employees to any position or to another department or direction.
  4. View an employee's profile with all their affections during their journey.
  5. View an employee's resume.
  6. Search for an employee file

Simulation 2


    This application manages the monitoring of files entering an institution until the file is completed.


  1. It allows you to register all the users who will intervene in the processing chain.
  2. To save the file upon arrival with the current status then transfer it to the processing service. once processed by the latter the status changes to processed status and ready to be delivered.
  3. When we deliver the file, the status changes to Delivered, then we can print a receipt for the client.
